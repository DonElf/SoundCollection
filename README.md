# SoundCollection
A Unity C# script to make managing/playing audio clips from a group quick &amp; easy,

To use, first create a Sound Collection asset.  
<img width="847" height="385" alt="image" src="https://github.com/user-attachments/assets/1311e2ca-b048-4f08-ac53-b3f41e33ee56" />

Then, select the audio clips and collection, and click "Add to sound collections".  
<img width="868" height="704" alt="image" src="https://github.com/user-attachments/assets/97acc974-f91c-4385-99e0-71851ebf9d4a" />

Alternatively, clicking on the sound collection will allow you to edit the list of clips in the inspector.  
<img width="553" height="288" alt="image" src="https://github.com/user-attachments/assets/8f1a8843-a64e-4dbd-93dc-1ec4ac45685e" />

Options are available to add audio clips recursively, or to add dupliactes (just in case you need that). Recursive searching caches folders for good validation optimisation.
